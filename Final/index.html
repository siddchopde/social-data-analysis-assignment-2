* Summarize the discussion of Graphical Analysis and Presentation Graphics on pp. 68-69 in your own words.

Basically, there are two main reasons to visualize the data: 1)Investigate the data and learn more about it (Graphical Analysis) 2)Present the data to others forcommunicating the information about the data (Presentation Graphics) For the first one the analyst is likely to behave like a detective and visualize the data from many angles. The analyst might not put all the labels etc. since he/she knows what the plot is about. Of course some techniques which would make it easy to read the data is always helpful. On the other hand, when it comes to the presentation, it is really crucial to prepare a plot that is easily readible. Thus here font sizes, colors, length of the axes all become important. It is important to think that the data will be communicated using the visualization so it is best to keep away from all frictions.
